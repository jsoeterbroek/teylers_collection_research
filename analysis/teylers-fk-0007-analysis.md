## Object Description

This is a spirit level calibration device made by Gebroeders Caminada, Rotterdam, a prominent Dutch instrument-making firm of Italian origin (active 1856–c. 1920). The device is a bench-top apparatus used to test and calibrate spirit level vials used in surveying leveling instruments. Its production period (1882–1898) coincides with the Netherlands' Eerste Nauwkeurigheidswaterpassing (First Precision Leveling Survey, 1875–1885), which demanded rigorously calibrated instruments.

## Key Findings Not in Teylers' Record

### 1. GEBR. CAMINADA: SIGNIFICANT INSTRUMENT-MAKING DYNASTY

The Caminada family were Italian immigrants who became one of the Netherlands' notable instrument-making dynasties:

- **Patriarch:** Carolus Anthonij Caminada (1763–1828), barometer maker from Milan, moved to Delft in 1788
- **Founders of the Rotterdam firm:** Carel Antonius Caminada (1829–1921) and Johannes Antonius Caminada (1830–1887) established "Gebr. Caminada" in Rotterdam in **1856**
- **Separate Hague branch:** Pieter and Hendricus Johannes Caminada established "Gebr. P. & H. Caminada" in The Hague in 1867
- **Products:** Barometers, nautical instruments, surveying/leveling instruments, drawing instruments, telegraph equipment (Morse keys), optical instruments, and later radio equipment
- **Active period:** 1856–c. 1920 (some sources suggest until the Rotterdam bombardment of May 1940)
- **Jubilee publication:** *Vijftig-jarig bestaan van de fabriek Firma Gebr. Caminada Rotterdam 1856–1906* (held at IISG Amsterdam)

This biographical context — establishing Caminada as a significant firm in Dutch instrument-making history — is entirely absent from Teylers' record.

### 2. HISTORICAL CONTEXT: THE FIRST PRECISION LEVELING SURVEY

The production dates (1882–1898) place this device squarely in the era of the **Eerste Nauwkeurigheidswaterpassing** (First Precision Leveling Survey, 1875–1885), conducted by Rijkswaterstaat to establish and verify height reference marks across the entire Netherlands. This survey:

- Led to the formalization of the **Normaal Amsterdams Peil (NAP)** — the Amsterdam Ordnance Datum, still the reference datum for Northwestern Europe
- Demanded extremely precise leveling instruments
- Required rigorous calibration of those instruments

A spirit level calibration device from this exact period represents the **metrological infrastructure** that made this nationally significant geodetic project possible. This context is absent from Teylers' record.

### 4. FUNCTION NOT FULLY EXPLAINED

The title "Toestel voor het ijken van waterpassen" is accurate but the catalog provides no explanation of what the device does. It is a **precision bench fixture** that:

1. Holds a spirit level vial in the ivory V-supports
2. Allows the operator to apply precise angular tilts using the fine-adjustment wheel
3. Tests whether the vial reads true horizontal correctly
4. Verifies the sensitivity of the vial (angular change per unit of bubble displacement)
5. Checks for systematic errors before vials are installed in surveying instruments

This is a **second-order precision instrument** — a tool for testing tools — which is particularly significant from a history-of-science perspective.
